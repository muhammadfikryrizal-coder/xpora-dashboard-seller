# XPORA Seller Dashboard

A React dashboard UI for monitoring UMKM production, quality control, orders, and export activity.

## Tech Stack

- React (Vite)
- Recharts
- Lucide React

## Project Structure

```text
xpora/
|- public/
|  |- vite.svg
|- src/
|  |- assets/
|  |- components/
|  |- App.jsx
|  |- index.css
|  |- main.jsx
|- .gitignore
|- index.html
|- package.json
|- vite.config.js
```

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open in browser:

```text
http://localhost:5173
```

## Available Scripts

- `npm run dev`: Start local development server.
- `npm run build`: Build production assets.
- `npm run preview`: Preview production build locally.

## Main Dependencies

- `react`
- `react-dom`
- `recharts`
- `lucide-react`

## Notes

- Main UI entry is `src/App.jsx`.
- This project currently focuses on a single dashboard page implementation with multiple sections and views.
